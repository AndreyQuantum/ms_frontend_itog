# AI Chat Client (ms_frontend_itog)

Современный и отзывчивый интерфейс для общения с нейросетями (Google Gemini) в режиме реального времени. Проект написан на React с использованием TypeScript, Vite и глобального хранилища Zustand.

## 🚀 Особенности и функционал

- **Динамическая авторизация**: Ключ API вводится непосредственно в интерфейсе приложения и надежно сохраняется в браузере.
- **Потоковая передача (Streaming)**: Поддержка стриминга для получения ответов от нейросети в реальном времени.
- **Подсветка кода (Markdown)**: Автоматический парсинг Markdown с подсветкой синтаксиса при помощи `react-markdown`.
- **Сохранение истории**: Чаты и настройки сохраняются в `localStorage`, что позволяет сохранять контекст между сессиями.
- **Умная обработка ошибок**: Понятные уведомления на русском языке, если API ключ недействителен или возникли проблемы со связью.
- **Современный UI**: Плавные анимации, индикация набора текста и адаптивный дизайн для мобильных устройств.

## ⚙️ Инструкция по запуску локально

### 1. Предварительные требования
- **Node.js** (версия 18.0 или выше)
- Ключ **Google Gemini API** (получите его в [Google AI Studio](https://aistudio.google.com/)).

### 2. Установка
```bash
npm install
```

### 3. Запуск
```bash
npm run dev
```
После запуска откройте [http://localhost:5173/](http://localhost:5173/).

### 4. Ввод ключа
При первом входе приложение попросит вас ввести **Gemini API Key**. 
- Ключ сохраняется локально в вашем браузере.
- Вы всегда можете изменить или обновить ключ в **Настройках** (иконка шестеренки).

## 💡 Примеры работы

### Сценарий 1: Первичная настройка
1. При запуске введите свой ключ `AIza...` в появившееся окно авторизации.
2. Нажмите "Начать". Если ключ валиден, вы сразу попадете в интерфейс чата.

### Сценарий 2: Изменение параметров модели
1. Нажмите на иконку настроек (шестеренка).
2. Вы можете изменить не только API ключ, но и параметры генерации: `Temperature`, `Top P`, `Max Tokens` или задать `System Prompt` для изменения стиля ответов ассистента.

### Сценарий 3: Остановка генерации
Если ответ становится слишком длинным, нажмите на кнопку **Стоп** (красный квадрат), чтобы мгновенно прервать поток данных.
